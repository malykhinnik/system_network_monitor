The monitor program was created using the shell script.<br/>
If it runs without parameters, it will show system and network data:<br/>
  - Internet Connectivity<br/>
  - OS and chassis type<br/>
  - Host information<br/>
  - Memory usage<br/>
  - CPU utilization<br/>
